Распаковываем архивы в терминале Ubuntu

Обычно встречаются архивы имеющие расширения .gz, .tar, .zip, .rar, .bz2.

Архив zip:

zip file.zip file - создать архив file.zip состоящего из файла с названием file;

unzip file.zip - распаковать архив file.rar в текующую директорию.

Архив gzip:

gzip file - создать архив file.gz состоящего из файла с названием file;

gzip -d file.gz - распаковать архив file.gz в текующую директорию.
