Инструкция по работе с WinSCP

sFTP (SSH File Transfer Protocol) - протокол прикладного уровня, для передачи данных, подобно протоколу ftp, только в зашифрованном виде. В качестве базового протокола используется безопасный протокол SSH.

По умолчанию на всех тарифах нашего хостинга доступ по протоколу SSH выключен.

Для того, чтобы включить доступ по SSH на вашем аккаунте обратитесь, пожалуйста, в службу технической поддержки. Включение протокола SSH абсолютно бесплатно и займёт несколько минут.Для того, чтобы подключиться по протоколу sFTP можно использовать SSH-клиент WinSCP.Скачать WinSCP можно здесь

WinSCP - это графический клиент SFTP для Windows с открытым исходным кодом. Он также поддерживает протокол SCP. Предназначен для защищённого копирования файлов между компьютером и серверами, поддерживающими эти протоколы.

Для подключения по SSH протоколу используются те же параметры, что и для подключения по основному FTP-доступу.

И так, скачиваем и устанавливаем WinSCP.

При первой загрузке Вы увидите окно следующего вида

Вводите имя сервера, "Host Name". Имя сервера указано в Панели Управления на главной странице. Имя пользователя и пароль ("User Name", "Password") - это Ваш логин и пароль для входа в Панель Управления.

Вы можете сразу подключиться, нажав "Login", или для удобства сохранить сессию, нажав "Save". При сохранении сессии будет предложено ввести имя сессии и сохранять или нет пароль.

На вкладке "Stored Sessions" Вы можете видеть сохранённые сессии. Вы можете выбрать сохранённую сессию в списке и нажать "Login" для подключения.

На вкладке "Preferences" Вы можете настроить вид программы после подключения. Вы можете выбрать интерфейс в стиле Norton Commander или в стиле проводника Windows Explorer. По умолчанию выбран первый.

После подключения, если Вы выбрали интерфейс в стиле Norton Commander, Вы увидите примерно следующее.

Этот интерфейс более удобен для управления с клавиатуры, им можно пользоваться вообще без помощи мыши. Он также позволяет выполнять основные операции быстрее и более наглядно.

После подключения, если Вы выбрали интерфейс в стиле проводника Windows Explorer, Вы увидите примерно следующее.

Этот интерфейс хорошо известен всем пользователям ОС Windows.

Завершить сессию Вы можете из вкладки "Session", выбрав "Disconnect", или воспользовавшись комбинацией горячих кнопок.

WinSCP позволяет работать с файлами прямо на сервере и легко справится с большинством задач, например, Вы можете изменить владельца, группу или права на файлы, как представлено на изображениях ниже.
