Включаем пользователя root в Ubuntu

май 5, 2013 by GUIDETO

По умолчанию, зайти в систему Ubuntu под пользователем root невозоможно. Если же пользователь rootнеобходим, то чтобы его включить, надо задать пароль. Откройте терминал и выполните команду:

sudo passwd root

Далее вам надо будет ввести пароль вашего текущего пользователя, под которым вы вошли в систему, а затем задать пароль для пользователя root.

Пароль для пользователя root создан.  Теперь надо сделать возможность локального входа root в нашу систему. Кликаем Система > Администрирование > Окно входа в систему На вкладке «Безопасность»отмечаем галочку «Разрешить локальный вход администратора системы». Хочу отметить, что не надо заходить в систему по рутом, для выполнения повседневных вещей. Будь то посидеть в аське или посмотреть фильм. Используйте учётную запись root лишь в том случае, если это необходимо.
