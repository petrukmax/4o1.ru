ASTERISKNOW меняем стандартные пароли

Смена стандартных паролей

После установки сервера, обновляемся и начинаем настраивать нашу программную-АТС.

Нам необходимо сменить стандартный пароль

[admin@asteriamroot]# mysql -u freepbx -p

Вводим стандартный пароль: fpbx

SET PASSWORD FOR freepbx@localhost=PASSWORD('новый-пароль-от-mysql ');

по аналогии в /etc/amportal.conf

AMPDBPASS=новый-пароль-от-mysql

Меняем пароль администратора Asterisk Manager Interface и Asterisk Recording Interface:

AMPMGRPASS=пароль-администратора-amportal

ARI_ADMIN_USERNAME=admin

ARI_ADMIN_PASSWORD=новый-пароль-ari

Если последних двух строк нет, то добавляем их в конфигурационный файл.

Далее меняем пароль администратора Asterisk Manager Interface еще в одном месте:

[admin@asteriamroot]# nano /etc/asterisk/manager.conf

secret = пароль-администратора-amportal

Меняем пароль от базы данных для системы отчетов CDR:

[admin@aasteriamroot]# nano /etc/asterisk/cdr_mysql.conf

password = новый-пароль-от-mysql

Изменить пароль администратора WebGUI FreePBX в соответствующем разделе интерфейса.

После всех манипуляций с паролями можно продолжить дальнейшую настройку.

Заходим в интерфейс FreePBX в раздел Module Administration и устанавливаем недостающие модули
