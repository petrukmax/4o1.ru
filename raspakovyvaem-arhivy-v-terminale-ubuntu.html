Распаковываем архивы в терминале Ubuntu

август 29, 2012 by GUIDETO

Обычно встречаются архивы имеющие расширения .gz, .tar, .zip, .rar, .bz2.

Архив zip:

zip file.zip file - создать архив file.zip состоящего из файла с названием file;

unzip file.zip - распаковать архив file.rar в текующую директорию.

Архив gzip:

gzip file - создать архив file.gz состоящего из файла с названием file;

gzip -d file.gz - распаковать архив file.gz в текующую директорию.

Архив bzip2:

bzip2 file - создать архив file.bz2 состоящего из файла с названием file;

bzip2 -d file.bz2 - распаковать архив file.bz2 в текующую директорию.

Архив tar:

tar -cf file.tar file - создать архив с именем file.tar состоящего из файла с названием file;

tar -xvf - распаковать архив file.tar в текующую директорию, с подробным списком извлекаемых файлов.

Архив rar:

rar a file.rar file - создать архив file.rar состоящего из файла с названием file;

rar e file.rar - распаковать архив file.rar в текущую директорию.

rar e file.rar path - где path путь куда разархивировать

Можно все эти операции забить в привычные для вас названия команд через alias:

alias tar='tar -xvf' и т.д.

Если прогамма не установлена, установите её

sudo apt-get install p7zip-full
